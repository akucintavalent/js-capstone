## Built With

- HTML&CSS, JavaScript
- Webpack

## Using the project locally

- Clone the project from GitHub.
- Run the following commands as listed in your terminal:
 - `npm install`
 - `npm run build`
 - Open `dist/index.html` or run `npm start`
